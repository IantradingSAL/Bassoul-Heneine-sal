Employees
Each employee is a person who logs in. Set their role here; this controls which pages they can open and edit. To create the matching login, add the user in Supabase → Authentication → Users, then paste the resulting User UUID into the "Auth User ID" field below.

Roles & page access
Each role has a permission level on every page: Edit (full access), View (read-only — buttons disabled, forms locked), or None (page is blocked). Click a role to expand the matrix. The built-in Manager role always has Edit on everything, regardless of what's set here.

System configuration
Stored in public.system_settings. The Customer Feedback module reads these to send email / WhatsApp / SMS notifications when records are created or reassigned.

Brevo (email + WhatsApp)
Used for outbound emails and WhatsApp messages. Get your API key from Brevo → SMTP & API.

Twilio (SMS)
Optional — only needed if you want SMS in addition to (or instead of) WhatsApp. The phone number must be in E.164 format including the country code.
